## About 
A web application for blogging

## Instructions to set up the backend server

1. **Clone the repository**  
    ```bash
    git clone <repository-url>
    ```   
2. **Open terminal and navigate inside the below directory**
   ```bash
    cd cvwo-backend-go/cmd/server
    ```

3. **Run the backend server (ensure frontend setup is complete)**  
    ```bash
   go run .
    ```
